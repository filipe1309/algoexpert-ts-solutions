# Insertion Sort
> Source: https://www.algoexpert.io/questions/insertion-sort  
> Difficulty: easy  
> Category: Sorting
---

Write a function that takes in an array of integers and returns a sorted version
of that array. Use the Insertion Sort algorithm to sort the array.

### Sample Input
```javascript
array = [8, 5, 2, 9, 5, 6, 3]
```

### Sample Output
```javascript
[2, 3, 5, 5, 6, 8, 9]
```
